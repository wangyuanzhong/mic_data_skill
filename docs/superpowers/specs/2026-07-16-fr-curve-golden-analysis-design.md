# Design: 频响曲线金标分析 Skill（fr-curve-golden-analysis）

**日期:** 2026-07-16  
**状态:** 步骤 0–5 主流程已定稿（含金标/辅标排名、双轨标准、报告合同）；实现期再补脚本命令与 layout 解析细节；样例不进仓库  
**仓库:** `mic_data_skill`  
**Skill 路径（计划）:** `.cursor/skills/fr-curve-golden-analysis/`

---

## 1. 目标

借助 Cursor Agent Skill，对一批（或多产品）频响测量数据完成：

1. 格式不确定下的批量理解与映射确认  
2. **两套独立标准：** 灵敏度标准 + 曲线（频响形状）标准  
3. 曲线金标 = 排名最优；辅标 = 往后顺延若干  
4. 灵敏度按绝对灵敏度 **0.5dB 一档**划分展示  
5. 可复现过程工作簿 + **一产品一结果报告**

**不是**再做一个完整桌面分析软件；而是把产线经验写成 Agent 必走的 SOP。

---

## 2. 核心分工（方案 2′）

| 角色 | 职责 |
|------|------|
| 用户 | 门禁确认；确认金标/辅标（默认按排名，可改） |
| AI（Agent） | 探查布局、映射沟通、奇异值提名、解读与简报 |
| 脚本 | 可重复数学：均值、归零、包络排名、灵敏度分档等 |

```
乱格式原始文件（只读）
    → AI 探查 + 门禁确认映射（会话内理解，不落标准曲线库）
    → 门禁确认产品参数
    → 算均值 → 奇异值门禁 → 剔除后重算均值
    → 脚本写入「一产品一过程工作簿」
    → 确认金标/辅标（曲线：第1=金标，其后=辅标；灵敏度另套）
    → 每个产品一份报告（report.md + figures；无总览页、无门禁6）
```

**铁律：**

1. 不落「标准化曲线库」文件；标准化只在过程中进行。  
2. 脚本出数；AI 出叙述。  
3. 未过必选门禁不得假装完成。  
4. **一产品一报告**；多产品隔离；不要多产品总览页。  
5. **灵敏度标准**与**曲线标准**互不绑定。

## 4. 步骤 1：批量探查与门禁 1

### 4.1 入口

用户给**文件夹** → 枚举候选文件 → 批量探查。

### 4.2 布局目录

| 布局代号 | 识别要点 | 典型样例 |
|----------|----------|----------|
| `batch_wide_fr` | 纵轴频率；横轴样机号；常有 Product / Units / `*1000Hz*` / `Curves` | `456_FR.xlsx` |
| `spl_vertical_calsod` | `CALSOD` 头；频率、幅度、相位 | `950138.SPL` |
| `single_horizontal_tsv` | 文本冒充 `.xls`；一行频率一行幅度 | `smile.xls` |
| `unknown` | 对不上或混杂（如 `chaos_blackbox_50`） | 多种 |

多布局并存 → 按子组映射，门禁1分组展示。

### 4.3 探查剧本

文件清单、布局、频率/幅度/样机启发式、批内一致性、风险点；产出预览与拟定映射。

### 4.4 门禁 1（UI 友好）

用表展示理解，禁止干问「哪列是频率」：表 A 清单、A′ 布局、B 预览、C 映射、D 曲线清单。  
未确认 → 禁止开算。

### 4.5 标准化含义

会话内按映射理解即可。映射摘要写入该产品 **`process.xlsx` 的 `manifest` + 报告 §1/§2`**，不另建标准曲线库文件。

## 5. 多产品编排

1. 识别产品列表。  
2. 多个 → 产品拆分表确认。  
3. **for each 产品** 跑步骤 2→5。  
4. 禁止混产品算均值/金标。  
5. **每产品**独立 `process.xlsx` + `report.md` + `figures/`；**不写** `index.md` 总览。

---

## 6. 步骤 2：产品参数与门禁 2

确认（可预填）：

- 产品名 / 批次  
- **分析频段**（默认 20–20000 Hz）——用于曲线包络  
- 1kHz 灵敏度策略：`batch_mean` / `design_value`（及设计值）——仅灵敏度轨道  
- 单位  
- 曲线辅标个数（默认如 2～3）

不要让用户填「贴均值公差」当金标门槛。

**0.5dB 的含义：** 指 **绝对灵敏度**（主看 1000Hz）按 **0.5dB 一档**划分样品。  
**不是**曲线包络步长（曲线用 ±0.1、±0.2… 收紧，见 §8）。

---

## 7. 步骤 3：均值 → 奇异值 → 重算

### 7.1 顺序

```
未剔除全集 → 平均曲线
  → 提名奇异值 → 门禁3
  → 剔除 → 重算平均曲线
  → 曲线包络排名 / 灵敏度分档
```

### 7.2 豁免（不算奇异值）

| 条件 | 规则 |
|------|------|
| &lt; 100 Hz | 不算 |
| &gt; 15000 Hz | 不算 |
| 高 Q | **Q ≥ 10** 视为特别尖锐，不算；标「豁免（高Q≥10）」 |

**Q：** 峰/谷中心 \(f_0\) 与约 −3dB 带宽 \(\Delta f\)，\(Q=f_0/\Delta f\)。无法稳定估 Q 则不给高 Q 豁免。

### 7.3 门禁 3

表：sample_id、建议、理由、用户决定。未确认不得出曲线金标结论。

## 8. 步骤 4：双轨标准 + 过程工作簿

### 8.0 两套标准（互不绑定）

| 轨道 | 内容 |
|------|------|
| **A. 灵敏度** | 绝对灵敏度（1000Hz）；中线=批均值或设计值；按 **0.5dB 一档**划分；可另有灵敏度侧代表，**与曲线金标无关** |
| **B. 曲线** | 1kHz 归零后形状；相对均值做减法；用分析频段 **±包络** 排名 |

### 8.1 曲线金标 / 辅标（最贴均值）

1. 剔异后样本，1kHz 归零。  
2. 各频率点简单平均 → 平均曲线。  
3. 样本 − 平均 → 偏差曲线。  
4. 在分析频段内，找能框住该样本的最紧包络半宽 δ（±0.1 能卡住就不要用 ±0.2 当它的等级；步进 0.1dB）。  
5. **排名：** δ 越小越好；同 δ 用偏差 RMS 破平局。  
6. **金标** = 第 1；**辅标** = 第 2、3… 再选几个。

### 8.2 灵敏度 0.5dB 分档

- 各样 1000Hz 绝对灵敏度。  
- **相对中线**，按 **0.5dB 一档**划分。  
- 报告主图/附录分档图画的是这套灵敏度分档。

### 8.3 「脚本接口」说明（非业务概念）

实现时一条命令：告诉计算器「原文件在哪、manifest 里映射/参数/剔除是什么、结果写到哪个 xlsx」。  
设计阶段不另造协议；映射以 **报告 + process 表** 为准即可。

### 8.4 `process.xlsx` sheets

产出目录与分析根**同级**（分析根父目录下）：`<产品>_<时间>/process.xlsx`（禁止建在仓库 `output/`）。

| Sheet | 内容 |
|-------|------|
| `manifest` | 源文件、映射摘要、参数、剔除、版本 |
| `sensitivity_1000hz` | 读数、中线、距中线、0.5dB 档 |
| `mean_before_outlier` | 过程追溯 |
| `outlier_review` | 过程追溯 |
| `mean_after_outlier` | **报告用均值** |
| `curve_rank` | 最紧 δ、排名；金标/辅标 |
| `sensitivity_standards` | 灵敏度侧汇总 |
| `excess_detail` | 可选 |

## 9. 步骤 5：结果报告合同

### 9.1 原则

- 无门禁6；无多产品总览。  
- 一产品一报告。  
- 正文只要剔异重算后的**结果**。  
- 双轨分章，互不要求「同时最优」。

### 9.2 交付物

`process.xlsx` + `report.md` + `figures/`（每产品一套）。

### 9.3 章节（固定顺序）

```markdown
# {产品} 频响测量分析报告
## 1. 运行追溯
## 2. 数据与映射摘要
## 3. 分析参数
## 4. 灵敏度标准（1000Hz / 0.5dB 分档）
## 5. 曲线标准（金标与辅标）
## 6. 产品问题线索
## 7. 结论与建议
## 附录 A. 过程表索引
## 附录 B. 灵敏度 0.5dB 分档图
```

### 9.4 要点

- **§4：** 表 + 主图 `fig_sensitivity_0p5dB_overview.png`（全家福 + 0.5dB 档线）  
- **附录 B：** 按档分组图（选 C）  
- **§5：** 金标第1、辅标随后；图 `fig_curve_golden_overlay.png`；列出各样 δ  
- **§7：** 灵敏度一句 + 曲线金标/辅标一句  

数字只来自过程表；缺图 = 未完成。
